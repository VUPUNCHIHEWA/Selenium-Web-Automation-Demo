# Web UI Test Automation - SauceDemo

### 📌 Project Overview
This project demonstrates automated functional testing for an e-commerce platform (**SauceDemo**) using **Selenium WebDriver** and **Java**. The goal is to automate critical user flows such as authentication and dashboard navigation to ensure system reliability.

### 🛠 Tech Stack
* **Language:** Java
* **Automation Tool:** Selenium WebDriver (v4.18.1)
* **Build Tool:** Maven
* **Driver Management:** WebDriverManager
* **Testing Site:** [SauceDemo](https://www.saucedemo.com/)

### 🚀 Key Features & Implementation
* **End-to-End Flow:** Automated the complete login process and verified successful redirection to the products page.
* **Locator Strategies:** Utilized various locators including **ID, Name, and CSS Selectors** for robust element identification.
* **Synchronization:** Implemented Wait mechanisms to handle dynamic web elements and ensure script stability.
* **Clean Code:** Structured the project using Maven for efficient dependency management.
